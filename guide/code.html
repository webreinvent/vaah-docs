# Naming conventions

# Variables

  • Meaningful / Predictable
  • Small letters
  • 2 letter much be joined with underscore (_)

$my_name = "John";

$var = "John";
$n = "John";

# Constants

  • Meaningful / Predictable
  • All capital letters
  • 2 letter much be joined with underscore (_)

Wrong Examples:

define('db_HOST', 'localhost'); 
define('DBUSER', 'db_user'); 
define('DB_pASS', 'password$%@#');
define('db_name', 'db_app');
