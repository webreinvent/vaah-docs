# Generators

Improve your productivity via quick command generator

Make sure you have install VaahCLI from following command:

```shell
npm i vaah -g
```


Use following command to generate `module`:

```shell
vaah cms:m
```


Use following command to generate `theme`:

```shell
vaah cms:t
```

Module Theme Routes

| ------------ | ---- | -------------------------------------------- |
